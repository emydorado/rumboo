# Rumboo Backend

Scaffold del backend para la API REST de Rumboo (TypeScript + Express + Supabase).

Scripts:

- `npm run dev` - arranque en desarrollo con `ts-node-dev`
- `npm run build` - compila a `dist/`
- `npm start` - ejecuta `dist/index.js`

Configurar variables en `.env` a partir de `.env.example`.
